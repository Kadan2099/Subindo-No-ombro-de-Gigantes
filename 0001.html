O que é HMTL?

HTML ou Hypertext Markup Language (Linguagem de marcação de Hiper texto) usa um sistema de marcação de elementos que representam um conteúdo específico, falando de forma mais natural se trata de um sistema em que demarcamos através de tags qual o conteúdo temos delimitado, mas lembrando que não delimitamos o como se apresenta, ou seja ao utilizar a tag <h1> na verdade estamos delimitando onde se está o título, mas não decidimos ainda se o título será Maior ou menor, qual fonte usamos e qual a cor, esse trabalho o css se encarrega de fazer.

Html NÃO É UMA LINGUAGEM DE PROGRAMAÇÃO, é uma linguagem de marcação, portanto não possui estrutura de decisão, loopings variáveis ou temas que normalmente temos em C, Python e Javascript.

Existe algumas tags que ainda insistem em permitir edições como:

  • <font>: Usada para estilizar o texto (fonte, cor, tamanho). Substituída por CSS.
  • <s> e <strike>: Para tachar texto (exibir um risco sobre ele). Substituídas por estilos de texto CSS.
  • <u>: Para sublinhar um texto. Substituída por estilos de texto CSS.
  • <center>: Para centralizar o conteúdo. Substituída por CSS para alinhamento.

Sintaxe Padrão

Normalmente as tags são utilizadas com a estrutura <Nome da tag-Abertura> Conteúdo </Nome da tag-Fechamento> um exempo é a tag <p></p> a tag de paragrafo que usamos constantemente no seguinte formato:

<p> Conteúdo,Conteúdo,Conteúdo,Conteúdo,Conteúdo,Conteúdo </p>

Existe exceções como as tags:

  • <meta> Fornece informações (metadados) sobre o HTML de uma página para navegadores, mecanismos de busca e redes sociais.
  • <link> Estabelece uma relação entre o documento HTML atual e um recurso externo
  • <input> Cria um controle de formulário interativo, permitindo que o usuário insira dados.

A página mais simples do universo


<!DOCTYPE html> 
<html lang="pt-br"> 
<head>  
  <meta charset="UTF-8">
  <title>Olá</title> 
</head> 
<body>
  <h1>Olá Mundo</h1>
  <p>O dia está lindo ou não?</p> 
</body>
</html>

  • <!DOCTYPE html>-- Define a versão usado no documento.
  • <html">-- Abre o conteúdo da página, nenhuma marcação pode ser utilizada após o fechamento desta tag, na abertura se determina qual a lingua em que a página se apresenta.
  • <head>-- Abrindo a seção do cabeçalho que não se apresenta na janela principal do navegador mas contém principalmente, informações sobre o documento Html, informações que são chamadas de metadados, através deste campo também se pode importar estilos de CSS e Scripts externos ao documento.
  • <meta> -- Dá ao navegador informações sobre o documento como o charset que declara o tipo de codificação(ASCII, UTF-8), mas isso não é obrigatório e assim como escrito acima não possui fechamento.
  • <title> -- Aqui se apresenta o titulo da página que aparece no topo na aba.
  • <body> -- Ou em português corpo, aqui se apresenta a parte do documento mostrado aos usuário dentro irá constar titulos paragrafos entre outras coisas a critério do programador.
  • <p> -- Paragrafo de texto.
