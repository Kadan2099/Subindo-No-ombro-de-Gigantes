Listas

Existm dois tipos de listas ordenadas (ordered lists) e não ordenadas (unordered lists) já usamos listas antes se tiver prestado atenção então vamos elaborar isso aqui.
Um ponto de atenção é que a lista sempre usa duas tags uma externa "pai" (<ol> ou <ul>) e outra "filha" <li> para delimitar item a item (vide código fonte).

Listas ordenadas

Como mencionado no resumo inicial, podemos usar a lista ordenada com a tag <ol> conforme exemplo abaixo;

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
Lista ordenadas possuem o parametro type (<ol type="1|a|A|i|I">) podendo assumir os seguintes valores:
  • 1 Valor Padrão com números decimais.
  • a Alfabético letras minusculas.
  • A Alfabético letras maiusculas.
  • i Números romanos letras minusculas.
  • I Números romanos letras maiusculas.

Por se tratar de ordenação os números podem ser manupulados manualmente incluindo a lista toda ou um item em especifico das seguintes formas:
Detro da tag <ol> incluimos o parametro start <ol start="5" > iniciando a lista com o valor parametrizado e dentro da tag <ol value="4" > com value incluimos o valor especificado ao item, qualquer item sequente seguirá a ordem apartir do último valor, exemplo:

  1. Começo em 5
  2. Subo para 6
  3. Vou para 4
  4. subo para 5
  5. De novo 6
Por último podemos reverter a ordem incluindo reversed dentro da tag <ol> , exemplo:
  1. Começo em 6
  2. Desço para 5
  3. Vou para 7
  4. Desço para 6
  5. De novo 5
