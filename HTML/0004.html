Âncoras e Hiperlinks

Aqui vamos ter que voltar uns passos atrás e deixar claro o que se fala sobre o que são parâmetros dentro de tags.

Parâmetros

Desde que inciamos, apenas uma vez usamos parâmetros dentro de tags, então eis a explicação;
Também conhecidos como atributos, parâmetros são palavras especiais que podem mudar o comportamento seguindo a estrutura < tag nome = "Valor">, existe varios tipos dos mais variados comportamentos para esses atributos, não se preocupe vamos explorar conforme avançamos.
Abaixo vamos intruduzir algumas.

Exemplo prático Âncora

Ancoras e links utilizam a tag <a> para representar suas referencias.
Gosto da ideia de considerar que tanto links quanto ancoras são do mesmo tipo considerando que de fato o são no quesito tag.
Como dito parametros tem a sintaxe < tag nome_do_parametro = "Valor"> com a tag <a> podemos utilizar os seguintes parametros .

  • href; usado para especificar o endereço para qual irá direcionar podendo ser um link absoluto ou relativo.
  • target; especifica onde se pode abrir o link podendo assumir os valores
    • _blank abre o conteudo em uma nova ATENÇÃO ao utilizar essa tag utilize junto rel="noopener noreferrer" por segurança Veja esse Artigo.
    • _parent abre o link no frame pai do atual (em breve estudaremos sobre frames).
    • _top abre o link no nível mais alto da janela, removendo todos os frames.
    • _self abre o link no mesmo local onde o usuário está.
  • title; da informações extras sobre um link, podendo exibir o mesmo que aparece quando se passa o mouse por cima igual o exemplo do href, inclusive esse tipo de parametro pode ser usado em várias tags diferentes.
  • download; diz ao navegador que o alvo será baixado o valor download="titulo" delimita o nome do arquivo baixado o navegador detectará automaticamente a extensão correta do arquivo e a adicionará ao nome (.img, .pdf, etc.). Se o valor for omitido, o nome original do arquivo será usado.
