Tipo de Documento (Doctypes)

Doctype, ou em português "tipo de documento", ajuda o navegador a entender qual é a versão do HTML utilizada, permitindo uma interpretação mais precisa da página. Isso é importante para garantir a correta renderização do site e pode influenciar na forma como ele é indexado por buscadores como o Google.

As declarações sobre o tipo de documento não são tags do html e pertencem ao topo do documento. Nesta parte declara a estrutura e declaração de vários tipos de documentos em HTML.

<!DOCTYPE html>

A declaração <!DOCTYPE html> deve sempre ser incluída no topo do documento HTML antes da tag <html> isso apartir do HTML5 se quiser pode pesquisar a fundo sobre as diferentes versões do HTML tenha em mente que a versão html 5 é a atualmente utilizada desde 2014, se quiser mais detalhes fique a vontade para pesquisar ou perguntar no perfil em que este trabalho estiver publicado.

As tags em HTML são case-insentive isso significa que indendente de escrever nas diferenças abaixo, todas são válidas:
<doctype html>
<dOCtyPe html>
<DocTYpe html>

Títulos e Sub-títulos

Há 6 tipos de tags que são consideradas "Títulos" em HTML sendo que:

  • <h1>Heading 1</h1> é o título mais importante
  • <h6>Heading 1</h1> é o título menos importante

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Sistemas de busca normalmente indexam páginas baseadas no conteúdo de seus títulos, para criar um site mais semantico para os navegadores é importante levar em consideração a estrutura dos títulos.
Em geral um documento deve ter apenas um h1 como o título principal tendo seus sub-títulos descendo na hierarquia conforme o número: <h1>Título principal</h1>
<h2>Sub-título 2</h2>
